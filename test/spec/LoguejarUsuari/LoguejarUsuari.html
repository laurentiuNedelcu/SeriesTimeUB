US 2.Test d'acceptació de loguejar un usuari

Com a client/usuari registrat Vull accedir a la meva sessió d'STUB Per poder gaudir dels serveis de l'aplicació STUB.

Criteri d'acceptació 1

En cas de que l'usuari estigui registrat quan estigui loguejant-se el sistema deixarà accedir a l'usuari a la seva sessió.

Nom real de l'usuari: None Nacionalitat: None Data de naixement: None Nom d'usuari: Nickname Contrasenya: Password Should be

Criteri d'acceptació 2

En cas de que el client no estigui registrat quan estigui loguejant-se el sistema respondrà amb un missatge d'error.
